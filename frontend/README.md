# shipper Lite Frontend

React-based frontend for document compliance checking.

## Tech Stack

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Data Fetching:** TanStack Query (React Query)
- **Icons:** Lucide React

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
# Development
npm run dev

# Build
npm run build

# Preview production build
npm run preview
```

## Code Quality

```bash
# Linting
npm run lint

# Formatting
npm run format

# Type checking
npm run typecheck
```

## Project Structure

```
src/
  components/     # Reusable UI components
  pages/          # Page components
  services/       # API client
  store/          # Zustand stores
  types/          # TypeScript types
  utils/          # Utility functions
```
